IC (Interactive Content) Specification

Version 1

Abstract

This document describes IC (Interactive Content), a versitile file format for the effecient transfer of related multimedia types under a single file. IC currently supports rich text, images, realtime website content, and relationships between content. Future versions plan to support direct embeds of other file types within a single IC file.
