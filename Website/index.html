IC (Interactive Content) Specification

Version 1

Abstract

This document describes IC (Interactive Content), a versitile file format for the effecient transfer of related multimedia types under a single file. Interactive Content allows displaying the relationships between rich text, images, and realtime website content. Future versions plan to support direct embeds of other file types within a single IC file.

Interactive Content aims to be a modern file format built for modern devices and data. Interactive Content is not meant to replace any existing method of data representation, but aims to build atop existing content to provide a more immersive experience.
